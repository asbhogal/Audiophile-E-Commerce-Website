<div align="center">

<h1>Audiophile E-Commerce Site</h1>

</div>

A fully functional ecommerce website showcasing a range of audio-related tech. Built using TypeScript, Next.js, Shopify, Storybook, Tailwind CSS, Shadcn, Playwright, React Hook Form, Zod and Husky. This repo is currently under development.
